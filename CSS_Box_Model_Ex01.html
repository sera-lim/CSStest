Exercise_01

  • h3 태그는 대표적인 block level element 이고, a 태그는 대표적인 inline element 이다.

    • CSS Box Model

    CSS(Cascading Style Sheets)는 마크업 언어가 실제 표시되는 방법을 기술하는 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이며, 레이아웃과 스타일을 정의할 때의 자유도가 높다.

  • 참고
    의미를 가지지 않으면서 단순히 영역을 구분하는 태그가 2개 있는데, div와 span.
    div는 block level element, span은 inline element라는 점이 다르다.
